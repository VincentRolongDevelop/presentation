Powering Operations with an Intuitive CMMS Frontend

Our enterprise-grade CMMS application, built with Flutter and Dart, simplifies complex data collection. It features a secure, modular architecture with robust role-based access control. This powerful frontend is driven by a Flask-based API, transforming your operational data into actionable intelligence through dynamic forms and advanced analytics.

Clean Architecture & Design

A Robust and Scalable Foundation

The application is structured in distinct layers to enhance maintainability and scalability. This design follows a strict dependency rule where outer layers depend on inner layers, but not vice versa, creating a robust and decoupled system.

UI / Views

Responsible for presenting data to the user and handling user interactions through Flutter widgets.

Controllers

Contains application-specific business logic and manages the state of the user interface.

Repositories

Manages data operations by abstracting data sources from the rest of the application.

Models

Represents the core business entities and data structures of the application.

Core Features & Modules

User Management Module

Comprehensive user administration, handling registration, profiles, role assignment, and environment-based access control.

Key Frontend Aspects:

  • Real-time Search: Instantly find users with live filtering by role and environment.
  • Efficient Pagination: Smoothly navigate large user lists with configurable page sizes.
  • Integrated Navigation: Seamlessly switch between user, role, and permission management screens.

Form Designer Module

Enables the creation, management, and configuration of dynamic forms for versatile data collection.

Key Frontend Aspects:

  • Debounced Search: Reduces server load and improves performance during form discovery.
  • State Preservation: Maintains form state during navigation for a seamless editing experience.
  • Automatic Refresh: Ensures data consistency by automatically updating lists after edits.

Form Submission & Viewing Modules

Handles the entire lifecycle of a form, from filling and submitting to browsing, searching, and managing submitted data.

Key Frontend Aspects:

  • Responsive Design: Adaptive UI components ensure a great experience on any screen size.
  • Client-Side Search: Provides instant discovery of forms and submissions without server lag.
  • State Management: Prevents accidental data loss with confirmation dialogs and progress tracking.
  • Manual Refresh: Pull-to-refresh functionality keeps submission data synchronized and up-to-date.

Analytics Module

Provides comprehensive reporting and data visualization, transforming raw data into business intelligence.

Key Frontend Aspects:

  • Centralized Dashboard: Presents key metrics and provides easy navigation to all analytics features.
  • Integrated Chart Builder: Allows for the creation of custom data visualizations on-the-fly.
  • Adaptive Layouts: Optimizes component sizing and spacing for perfect viewing on any device.
