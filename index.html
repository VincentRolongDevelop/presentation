PLG CMMS Frontend

A production-ready, enterprise-grade Computerized Maintenance Management System. Built with Flutter for complex, form-based data collection with robust role-based access control.

Technology Stack

Framework & Platform

Flutter SDK 3.6.1+: Powering a multi-platform experience for Android, Web, and iOS.

State Management

Provider Pattern: Efficient state management using `ChangeNotifier` for predictable logic.

API Communication

Dio HTTP Client: Robust REST API communication with smart retry capabilities.

Local Storage

Secure & Shared Storage: Using `Flutter Secure Storage` and `SharedPreferences`.

File Handling

Comprehensive media management with image picker, file picker, and path provider.

UI Components

A rich set of UI tools including form builders, color pickers, video players, and more.

Clean Architecture & Design

Layered Approach

The project follows a strict Clean Architecture with a clear separation of concerns, making it highly maintainable and scalable.

  • ✓Models: Data entities with JSON serialization.
  • ✓Repositories: Data access layer for API communication.
  • ✓Controllers: Business logic with Provider state management.
  • ✓Services: Infrastructure services (API, Auth).
  • ✓Views: UI layer organized by feature modules.

Dependency Injection

Utilizes the GetIt service locator pattern for efficient dependency injection. It employs sophisticated factory and singleton registrations to prevent state sharing conflicts, ensuring a robust and predictable application state.

Core Features & Modules

Authentication & User Management

JWT-based authentication with secure token storage and automatic refresh. Features granular Role-Based Access Control (RBAC) supporting Super Users, COOs, and regular users.

Comprehensive Form Management

The core of the application. Includes a form designer, submission handling with file attachments, form viewing, and management of various question types and answer formats.

User & Permission Management

Full CRUD operations for users with role assignment and environment-based organization. A detailed permission matrix links roles to specific permissions.

Analytics & Reporting

A powerful dashboard displays key metrics and performance indicators. Features advanced analytics with chart generation and data export functionality for comprehensive reporting.

Security & Performance

Robust Security Measures

  • ✓JWT Token Management: Automatic refresh and secure storage.
  • ✓Permission-based UI: Dynamically renders UI to prevent unauthorized access.
  • ✓API Request Interception: Handles authentication and errors gracefully.

Performance Optimizations

  • ✓Client-side Pagination: Intelligent fallback to server-side for large datasets.
  • ✓Data Caching: Configurable strategies to reduce network requests.
  • ✓Lazy Loading: Efficiently loads controllers and services on demand.
